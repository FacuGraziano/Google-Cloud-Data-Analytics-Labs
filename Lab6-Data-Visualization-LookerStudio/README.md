# Lab 6: Creación de Dashboards con Looker Studio

En este laboratorio, transformé datos crudos de BigQuery en visualizaciones interactivas utilizando **Looker Studio**. El objetivo fue facilitar la toma de decisiones mediante un panel de control (dashboard) de métricas de eCommerce.

### Objetivos del Lab:
* **Conectividad:** Conectar Looker Studio con BigQuery de forma directa.
* **Visualización de Datos:** Crear gráficos de series temporales, gráficos de barras y tablas dinámicas.
* **Interactividad:** Añadir controles de filtrado (por fecha, categoría y marca) para permitir al usuario explorar los datos.

### Tareas realizadas:
1. **Source Connection:** Configuré BigQuery como la fuente de datos principal.
2. **Diseño de Reporte:** Creé un tablero con las siguientes métricas:
   - Ventas totales a lo largo del tiempo.
   - Top de productos más vendidos por categoría.
   - Distribución de ventas por marca.
3. **Filtros Dinámicos:** Implementé selectores para que los usuarios puedan segmentar la información en tiempo real.

### Habilidades adquiridas:
- Diferencia entre dimensiones (campos cualitativos) y métricas (campos cuantitativos).
- Creación de campos calculados básicos para reportes.
- Mejores prácticas de diseño de dashboards (Data Storytelling).
