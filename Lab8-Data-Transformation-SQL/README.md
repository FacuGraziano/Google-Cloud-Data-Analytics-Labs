# Lab 8: Métodos de Transformación de Datos con SQL

En este laboratorio, apliqué técnicas de ingeniería de datos para transformar datasets crudos en tablas optimizadas para el análisis de negocio. El foco estuvo en la normalización, limpieza y enriquecimiento de datos directamente en BigQuery.

### Habilidades Técnicas Aplicadas:
* **Estandarización:** Uso de funciones de cadena (`UPPER`, `TRIM`) para uniformar datos categóricos.
* **Manejo de Nulos:** Implementación de `IFNULL` y `COALESCE` para asegurar la integridad referencial.
* **Ingeniería de Atributos (Feature Engineering):** Creación de nuevas métricas como el margen de ganancia y clasificación de días laborales/fines de semana.
* **Manipulación de Fechas:** Extracción y formateo de componentes temporales para análisis de estacionalidad.

### Impacto en el Negocio:
Estas transformaciones permiten que los usuarios finales (Analistas de Negocio) consuman datos ya validados y limpios, reduciendo el error humano en los reportes finales y mejorando la precisión de las métricas de rentabilidad.
