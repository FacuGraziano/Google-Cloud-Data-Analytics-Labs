# Pipeline de datos

El flujo de datos del proyecto es el siguiente:

1. Importación de datos desde Cloud Storage
2. Exploración de datos en BigQuery
3. Transformación de datos mediante SQL
4. Creación de tablas analíticas
5. Visualización de datos en Looker

Herramientas utilizadas:

- Google Cloud Storage
- BigQuery
- SQL
- Looker
